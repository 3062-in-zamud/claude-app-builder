# GDPR コンプライアンス・チェックリスト

**対象**: EU/EEA 居住者の個人データを処理するサービス

## 法的根拠の確認

- [ ] データ処理の法的根拠を特定した（同意/契約/正当利益/法的義務）
- [ ] 同意を法的根拠とする場合、明示的・自由意思による同意取得の仕組みがある
- [ ] 同意はいつでも撤回可能である
- [ ] 16歳未満（加盟国により13-16歳）のデータ処理には親権者の同意を取得

## プライバシーポリシー必須記載事項

- [ ] データ管理者の名称・連絡先
- [ ] DPO（データ保護担当者）の連絡先（該当する場合）
- [ ] 処理の目的と法的根拠
- [ ] 正当利益を根拠とする場合、その具体的な利益の内容
- [ ] 個人データの受領者（第三者サービス一覧）
- [ ] EU域外へのデータ移転の有無と安全措置
- [ ] データ保持期間または期間の決定基準
- [ ] データ主体の権利一覧（アクセス・訂正・削除・制限・可搬性・異議申立・自動処理）
- [ ] 同意の撤回権
- [ ] 監督機関への苦情申立権
- [ ] 契約履行のためにデータ提供が必要かの説明
- [ ] 自動的な意思決定・プロファイリングの有無

## データ主体の権利（実装必須）

| 権利 | 実装方法 | 対応期限 |
|------|---------|---------|
| アクセス権 | データエクスポート機能 or メール対応 | 1ヶ月以内 |
| 訂正権 | プロフィール編集機能 | 1ヶ月以内 |
| 削除権 | アカウント削除機能（data-deletion連携） | 1ヶ月以内 |
| 処理制限権 | データ処理の一時停止対応 | 1ヶ月以内 |
| データ可搬性 | JSON/CSV エクスポート機能 | 1ヶ月以内 |
| 異議申立権 | メール対応フロー | 1ヶ月以内 |
| 自動処理拒否権 | 人間によるレビューの代替手段 | 1ヶ月以内 |

## Cookie / トラッキング

- [ ] Cookie同意バナーを実装（cookie-consent スキル連携）
- [ ] 同意前にトラッキングCookieを設置しない
- [ ] Cookie は必須/分析/マーケティングにカテゴリ分けされている
- [ ] 同意はいつでも変更可能
- [ ] Google Consent Mode v2 に対応

## データ処理契約（DPA）

- [ ] Supabase との DPA を確認（https://supabase.com/legal/dpa）
- [ ] Vercel との DPA を確認
- [ ] Sentry との DPA を確認
- [ ] その他サブプロセッサーとの DPA を確認

## データ侵害対応

- [ ] データ侵害時の72時間以内の監督機関通知体制がある
- [ ] 高リスク侵害時のデータ主体への通知体制がある
- [ ] インシデント対応手順を文書化している

## 技術的・組織的措置

- [ ] データの暗号化（転送時: TLS、保存時: AES-256）
- [ ] アクセス制御（RLS、認証必須）
- [ ] ログ記録（アクセスログ、変更ログ）
- [ ] 定期的なセキュリティレビュー
