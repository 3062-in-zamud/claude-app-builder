# APPI（個人情報保護法）コンプライアンス・チェックリスト

**対象**: 日本国内でサービスを提供し、個人情報を取り扱う事業者
**根拠法**: 個人情報の保護に関する法律（2022年4月改正施行）

## 基本原則

- [ ] 利用目的をできる限り特定している
- [ ] 利用目的を本人に通知または公表している
- [ ] 利用目的の範囲内でのみ個人情報を利用している
- [ ] 不正な手段による個人情報の取得をしていない

## プライバシーポリシー必須記載事項

- [ ] 事業者の名称
- [ ] 利用目的（具体的に）
- [ ] 第三者提供の有無と提供先
- [ ] 開示等の請求手続き
- [ ] 苦情の申出先
- [ ] 個人情報の安全管理措置の概要

## 利用目的の特定

```markdown
## 個人情報の利用目的

当社は、以下の目的で個人情報を利用します:

1. **サービスの提供・運営**: アカウント管理、認証処理、サービス機能の提供
2. **カスタマーサポート**: お問い合わせへの対応、サポートの提供
3. **サービスの改善**: 利用状況の分析、新機能の開発
4. **お知らせの送信**: サービスに関する重要な通知、メンテナンス情報
5. **不正利用の防止**: 利用規約違反の検知、セキュリティの確保
```

## 第三者提供

- [ ] 第三者提供を行う場合、本人の同意を取得している
- [ ] オプトアウト方式の場合、個人情報保護委員会に届出している
- [ ] 委託先（Supabase/Vercel等）は「第三者」に該当しないことを確認

### 委託先の監督

| 委託先 | 所在地 | 委託内容 | 監督措置 |
|--------|--------|---------|---------|
| Supabase | 米国 | データベース保管 | DPA締結・暗号化 |
| Vercel | 米国 | ホスティング | DPA締結・暗号化 |
| Sentry | 米国 | エラー監視 | DPA締結・最小限データ |

## 海外移転（外国にある第三者への提供）

2022年改正で強化された要件:

- [ ] 本人の同意を取得している
- [ ] 移転先の国名を情報提供している
- [ ] 移転先国の個人情報保護制度を情報提供している
- [ ] 移転先における個人情報の保護に関する措置を情報提供している

```markdown
## 個人データの海外移転について

当サービスでは、以下の外国にある第三者にデータ処理を委託しています:

### 米国（United States）
- **対象サービス**: Supabase（データベース）、Vercel（ホスティング）
- **個人情報保護制度**: 連邦レベルの包括的な個人情報保護法はないが、
  カリフォルニア州CCPA等の州法、セクター別の規制が存在
- **保護措置**: 各サービスとのDPA（データ処理契約）締結、
  データの暗号化（転送時TLS 1.2以上、保存時AES-256）
```

## 開示等の請求

- [ ] 本人からの開示請求に対応する体制がある
- [ ] 訂正・追加・削除の請求に対応できる
- [ ] 利用停止・消去の請求に対応できる（data-deletion スキル連携）
- [ ] 第三者提供の停止請求に対応できる
- [ ] 請求手続きを公表している

### 請求対応フロー

```
1. 本人確認（メール認証 or 本人確認書類）
2. 請求内容の確認
3. 対応（原則2週間以内、最大1ヶ月）
4. 結果の通知
```

## 安全管理措置

### 組織的安全管理措置
- [ ] 個人データの取扱いに関する規程の整備
- [ ] 責任者の設置
- [ ] 取扱状況を確認する手段の整備

### 技術的安全管理措置
- [ ] アクセス制御（Supabase RLS）
- [ ] 通信の暗号化（HTTPS/TLS）
- [ ] 保存データの暗号化
- [ ] ログの記録

### 物理的安全管理措置
- [ ] クラウドサービス利用のため、クラウド事業者の措置に依存
- [ ] Supabase/Vercel のセキュリティ認証（SOC 2等）を確認

## 個人情報漏えい時の対応

2022年改正で義務化:

- [ ] 個人情報保護委員会への報告（速報: 概ね3-5日以内、確報: 30日以内）
- [ ] 本人への通知（速やかに）
- [ ] 対象: 要配慮個人情報の漏えい / 財産的被害のおそれ / 不正アクセス / 1,000件超

## 特定商取引法との連携

有料サービスの場合、特定商取引法に基づく表記も必要:
- 販売事業者名、代表者、所在地、連絡先
- 販売価格、支払方法、返品・キャンセル条件
