# CCPA/CPRA コンプライアンス・チェックリスト

**対象**: カリフォルニア州居住者の個人情報を収集するビジネス
**適用基準**: 年間売上$25M超 / 年間10万件超の消費者データ処理 / 個人情報販売で売上の50%超

> MVP段階では適用基準を満たさないケースが多いが、グローバル展開を見据えて
> 早期に対応しておくことを推奨。

## プライバシーポリシー必須記載事項

- [ ] 過去12ヶ月間に収集した個人情報のカテゴリ
- [ ] 各カテゴリの収集目的
- [ ] 個人情報のソース（直接収集・第三者から取得）
- [ ] 個人情報を共有する第三者のカテゴリ
- [ ] 消費者の権利一覧
- [ ] 権利行使の方法（最低2つ: ウェブフォーム + 電話番号 or メール）
- [ ] 過去12ヶ月間に個人情報を「販売」または「共有」したかの開示
- [ ] プライバシーポリシーの最終更新日

## 消費者の権利

| 権利 | 内容 | 実装方法 |
|------|------|---------|
| 知る権利 | 収集されるデータとその目的 | プライバシーポリシーに記載 |
| 削除権 | 個人情報の削除請求 | data-deletion スキル連携 |
| オプトアウト権 | 個人情報の販売/共有を拒否 | 「Do Not Sell」リンク |
| 訂正権 | 不正確な個人情報の訂正 | プロフィール編集機能 |
| データ可搬性 | 構造化形式でのデータ提供 | JSON/CSVエクスポート |
| 非差別の権利 | 権利行使による不利益禁止 | ポリシーに明記 |

## 必須 UI 要素

### 「Do Not Sell or Share My Personal Information」リンク

- [ ] ウェブサイトのフッターに配置
- [ ] ホームページから1クリックでアクセス可能
- [ ] クリック後の処理フローが実装されている

```tsx
// フッターに配置する例
<Link href="/privacy#do-not-sell">
  Do Not Sell or Share My Personal Information
</Link>
```

### プライバシー権行使フォーム

- [ ] 本人確認の仕組み（メール認証等）
- [ ] 45日以内の応答期限（最大90日まで延長可能）
- [ ] 無料で権利行使可能

## データ収集の分類

CCPA で定義されるカテゴリに分類:

| カテゴリ | 例 | 収集有無 |
|---------|-----|---------|
| 識別子 | 名前、メール、IPアドレス | [ ] |
| 商業情報 | 購入履歴、取引情報 | [ ] |
| インターネット活動 | ブラウザ履歴、検索履歴、サイト操作 | [ ] |
| 位置情報 | 大まかな位置（IPベース） | [ ] |
| 推測 | 嗜好、特性、行動パターン | [ ] |
| 機密個人情報 | 人種、宗教、健康情報 | [ ] |

## 第三者サービスの確認

| サービス | データ共有 | 「販売」に該当するか | 対応 |
|---------|----------|-------------------|------|
| Supabase | DB保存 | No（サービスプロバイダー） | DPA確認 |
| Vercel | ホスティング | No（サービスプロバイダー） | DPA確認 |
| Sentry | エラー情報 | No（サービスプロバイダー） | DPA確認 |
| Google Analytics | 行動データ | 要確認 | オプトアウト対応 |
