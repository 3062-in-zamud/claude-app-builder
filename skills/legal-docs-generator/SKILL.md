# legal-docs-generator: 法務ドキュメント生成

## 免責事項

**このツールが生成する文書はテンプレートです。法的効力については必ず弁護士・法律専門家にレビューを依頼してください。特に個人情報の取り扱いに関しては、お客様の所在国・事業内容に応じた法律が適用されます。**

## ワークフロー

### Step 1: インプット読み込みと対象地域判定

```
docs/requirements.md（サービス名・機能・対象ユーザー・対象地域）
```

**国別分岐ロジック**:

| 対象地域 | 追加要件 |
|---------|---------|
| 日本のみ | APPI（個人情報保護法）+ 特定商取引法表記（商用の場合） |
| EU | GDPR 準拠（データ可搬性・処理制限・自動処理拒否・Cookie同意） |
| US（カリフォルニア） | CCPA 準拠（Do Not Sell・削除権） |
| グローバル | GDPR + CCPA + APPI 全対応 |
| 13歳未満対象 | COPPA 準拠（親権者同意・データ最小化） |

`requirements.md` に対象地域が明記されていない場合、「日本のみ」をデフォルトとし、グローバル対応が必要か確認する。

### Step 2: プライバシーポリシー生成

`references/privacy-template.md` を元に以下をカスタマイズ:
- サービス名
- 収集データの種類（認証・DB 有無に応じて）
- 外部サービス（Supabase・Vercel・Sentry など）
- **データ削除権の明記**（data-deletion スキルとの連携案内）

#### GDPR セクション（EU 対象時に追加）

詳細は `references/gdpr-compliance-checklist.md` 参照。

```markdown
## EU 居住者の権利（GDPR）

以下の権利を行使できます:
- **アクセス権**: 保有する個人データのコピーを請求する権利
- **訂正権**: 不正確な個人データの訂正を請求する権利
- **削除権（忘れられる権利）**: 個人データの削除を請求する権利
- **処理制限権**: 特定の状況下でデータ処理の制限を請求する権利
- **データ可搬性の権利**: 構造化された機械可読形式でデータを受け取る権利
- **異議申立権**: ダイレクトマーケティング目的のデータ処理に異議を申し立てる権利
- **自動処理に関する権利**: プロファイリングを含む自動処理のみに基づく決定に服さない権利

データ保護担当者（DPO）連絡先: [メールアドレス]
監督機関への苦情申立権があります。
```

#### CCPA セクション（US カリフォルニア対象時に追加）

詳細は `references/ccpa-compliance-checklist.md` 参照。

```markdown
## カリフォルニア州居住者の権利（CCPA/CPRA）

- **知る権利**: 収集される個人情報のカテゴリと目的
- **削除権**: 個人情報の削除を請求する権利
- **オプトアウト権（Do Not Sell/Share）**: 個人情報の販売・共有を拒否する権利
- **非差別の権利**: 権利行使を理由にサービスの質を低下させない

「Do Not Sell or Share My Personal Information」リンクをフッターに配置すること。
```

#### APPI セクション（日本対象時に追加）

詳細は `references/appi-compliance-checklist.md` 参照。

```markdown
## 個人情報の取り扱いについて（個人情報保護法）

- **利用目的の特定**: 個人情報の利用目的をできる限り特定し、本人に通知・公表
- **第三者提供の同意**: 個人データを第三者に提供する場合は本人の同意を取得
- **海外移転の規制**: 外国にある第三者への個人データ提供時は、本人の同意を取得し、
  移転先の国名・個人情報保護制度を本人に情報提供
- **開示請求**: 本人は保有個人データの開示・訂正・利用停止を請求可能

※ Supabase（米国）、Vercel（米国）を利用するため、海外移転に関する説明が必要。
```

### Step 3: 利用規約生成

`references/terms-template.md` を元にカスタマイズ

### Step 4: Cookieポリシーページ生成

EU 対象時は必須。それ以外でもGoogleアナリティクス等を使用する場合は推奨。

```markdown
## Cookieポリシー

### Cookieとは
[Cookieの簡単な説明]

### 使用するCookieの種類

| カテゴリ | Cookie名 | 目的 | 有効期限 | 必須/任意 |
|---------|----------|------|---------|----------|
| 必須 | supabase-auth-token | 認証状態の維持 | セッション | 必須 |
| 分析 | _vercel_insights | アクセス解析 | 1年 | 任意 |
| マーケティング | _ga | Google Analytics | 2年 | 任意 |

### Cookie設定の変更
[cookie-consent バナーへの案内、ブラウザ設定での無効化方法]
```

### Step 5: 特定商取引法に基づく表記（日本向け商用サービス）

有料プランがある場合に生成。

```markdown
## 特定商取引法に基づく表記

| 項目 | 内容 |
|------|------|
| 販売事業者名 | [事業者名] |
| 代表者 | [代表者名] |
| 所在地 | [住所] |
| 電話番号 | [電話番号]（※ 請求時に遅滞なく開示） |
| メールアドレス | [メールアドレス] |
| 販売価格 | [プラン一覧ページ参照] |
| 支払方法 | クレジットカード |
| 支払時期 | 申込時に即時決済 |
| 商品引渡し時期 | 決済完了後、即時利用可能 |
| 返品・キャンセル | デジタルコンテンツのため返品不可。解約は随時可能 |
| 動作環境 | [対応ブラウザ] |
```

### Step 6: App Store 対応確認

- [ ] プライバシーポリシーが公開 URL にアクセスできる状態か
- [ ] 連絡先メールが記載されているか
- [ ] 最終更新日が記載されているか

### 出力ファイル

- `app/privacy/page.tsx`（プライバシーポリシーページ）
- `app/terms/page.tsx`（利用規約ページ）
- `app/cookies/page.tsx`（Cookieポリシーページ - EU対象時）
- `app/legal/page.tsx`（特定商取引法表記 - 日本向け商用時）
