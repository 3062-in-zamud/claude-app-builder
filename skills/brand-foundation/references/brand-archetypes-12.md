# ブランドアーキタイプ12原型

## 概要

Carl Jung のアーキタイプ理論に基づく12のブランド人格モデル。
プロダクトのブランドパーソナリティを一貫性を持って設計するために使用する。

## 12原型一覧

### 1. The Innocent（純真な者）
- **コアメッセージ**: シンプルで正しいものを
- **ボイス**: 明るい、楽観的、正直、シンプル
- **カラー傾向**: 白、パステルカラー、水色
- **適するプロダクト**: 子供向け、オーガニック、シンプルツール
- **ブランド例**: Coca-Cola、Dove、Nintendo

### 2. The Explorer（探検家）
- **コアメッセージ**: 自分の道を切り開く
- **ボイス**: 冒険的、大胆、自由、刺激的
- **カラー傾向**: アースカラー、カーキ、深緑
- **適するプロダクト**: 旅行、アウトドア、新しい体験
- **ブランド例**: The North Face、Jeep、Patagonia

### 3. The Sage（賢者）
- **コアメッセージ**: 知識で世界を理解する
- **ボイス**: 知的、分析的、客観的、教育的
- **カラー傾向**: ネイビー、深緑、グレー
- **適するプロダクト**: 教育、リサーチ、データ分析
- **ブランド例**: Google、BBC、TED

### 4. The Hero（英雄）
- **コアメッセージ**: 勇気で世界を変える
- **ボイス**: 力強い、自信に満ちた、挑戦的
- **カラー傾向**: 赤、ネイビー、ゴールド
- **適するプロダクト**: フィットネス、生産性、自己成長
- **ブランド例**: Nike、FedEx、BMW

### 5. The Outlaw（反逆者）
- **コアメッセージ**: ルールは壊すためにある
- **ボイス**: 大胆、挑発的、反骨、破壊的
- **カラー傾向**: 黒、赤、ダークカラー
- **適するプロダクト**: 破壊的イノベーション、カウンターカルチャー
- **ブランド例**: Harley-Davidson、Virgin、Diesel

### 6. The Magician（魔法使い）
- **コアメッセージ**: 夢を現実にする
- **ボイス**: 想像力豊か、ビジョナリー、変革的
- **カラー傾向**: 紫、ミッドナイトブルー、ゴールド
- **適するプロダクト**: テクノロジー、エンターテイメント、変革ツール
- **ブランド例**: Apple、Disney、Tesla

### 7. The Regular Guy（一般人）
- **コアメッセージ**: みんなと同じでいい
- **ボイス**: 親しみやすい、実用的、正直、飾らない
- **カラー傾向**: ブラウン、ベージュ、落ち着いた色
- **適するプロダクト**: 日用品、コミュニティ、実用ツール
- **ブランド例**: IKEA、Target、Gap

### 8. The Lover（恋人）
- **コアメッセージ**: 美しさと親密さ
- **ボイス**: 感性的、情熱的、エレガント、温かい
- **カラー傾向**: ピンク、赤、ゴールド、ワインレッド
- **適するプロダクト**: 美容、ファッション、グルメ、贈り物
- **ブランド例**: Chanel、Godiva、Victoria's Secret

### 9. The Jester（道化師）
- **コアメッセージ**: 今を楽しもう
- **ボイス**: ユーモラス、遊び心、軽快、面白い
- **カラー傾向**: オレンジ、黄色、明るいマルチカラー
- **適するプロダクト**: エンタメ、ゲーム、SNS、カジュアルアプリ
- **ブランド例**: M&M's、Old Spice、Dollar Shave Club

### 10. The Caregiver（世話人）
- **コアメッセージ**: 他者を守り助ける
- **ボイス**: 温かい、思いやり、安心感、包容力
- **カラー傾向**: 暖色系、パステル、青緑
- **適するプロダクト**: ヘルスケア、保険、NPO、子育て
- **ブランド例**: Johnson & Johnson、UNICEF、Volvo

### 11. The Creator（創造者）
- **コアメッセージ**: 想像を形にする
- **ボイス**: 革新的、インスピレーション、表現力
- **カラー傾向**: モノトーン + アクセントカラー
- **適するプロダクト**: デザインツール、アート、DIY、開発ツール
- **ブランド例**: Adobe、LEGO、Moleskine

### 12. The Ruler（支配者）
- **コアメッセージ**: コントロールと秩序
- **ボイス**: 権威的、自信、リーダーシップ、高級感
- **カラー傾向**: ネイビー、ゴールド、黒、シルバー
- **適するプロダクト**: 金融、エンタープライズ、高級品、管理ツール
- **ブランド例**: Microsoft、Mercedes-Benz、Rolex

## アーキタイプ選定フロー

### Step 1: 候補の絞り込み

以下の質問に答えてアーキタイプ候補を2〜3個に絞る:

| 質問 | 回答 | 示唆されるアーキタイプ |
|------|------|---------------------|
| ユーザーにどんな感情を抱いてほしい？ | | |
| 競合と比べてどんな存在でありたい？ | | |
| ブランドを人格化すると誰に近い？ | | |
| ペルソナの性格タイプは？ | | |

### Step 2: 整合性チェック

| チェック項目 | プライマリ候補 | セカンダリ候補 |
|-------------|-------------|-------------|
| ターゲットユーザーとの親和性 | ○/× | ○/× |
| 競合との差別化 | ○/× | ○/× |
| プロダクトの価値提案との整合 | ○/× | ○/× |
| ペルソナのデザイン嗜好との一致 | ○/× | ○/× |

### Step 3: ブランド要素への展開

選定したアーキタイプから以下を導出:

```
アーキタイプ: [名称]
ブランドボイス: [アーキタイプのボイス特性を具体化]
カラー方向性: [アーキタイプのカラー傾向 + ペルソナの嗜好]
トーン & マナー:
  - コピーライティング: [具体的なトーン]
  - UIテキスト: [具体的なトーン]
  - エラーメッセージ: [具体的なトーン]
```
