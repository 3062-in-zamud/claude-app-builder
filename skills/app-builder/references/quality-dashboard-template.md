## 生成方法

品質ダッシュボードは以下のコマンドの出力結果を集約して生成する:

```bash
# テストカバレッジ
npm run test:coverage 2>&1 | grep "All files"

# TypeScript
npx tsc --noEmit 2>&1 | tail -1

# ESLint
npx eslint src/ 2>&1 | tail -1

# npm audit
npm audit --audit-level=high 2>&1 | tail -5

# ビルド
npm run build 2>&1 | tail -5

# バンドルサイズ
# next build の出力から First Load JS を抽出
```
