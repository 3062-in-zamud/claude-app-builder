# TAM/SAM/SOM ボトムアップ算出方法

## 概要

市場規模をトップダウン（業界レポート）とボトムアップ（個別積算）の両方で算出し、
推定値の信頼性を高める。

## 定義

```
TAM (Total Addressable Market)
 └─ SAM (Serviceable Available Market)
     └─ SOM (Serviceable Obtainable Market)
```

- **TAM**: この製品カテゴリの全世界市場規模（理論上の最大値）
- **SAM**: 地理・言語・プラットフォーム等の制約を考慮した到達可能市場
- **SOM**: 1〜2年で現実的に獲得できる市場シェア

## ボトムアップ算出方法

### Step 1: 単位経済を定義する

```
月額単価 (ARPU) = ¥[金額]
年間単価 (ARPA) = ARPU × 12 = ¥[金額]
```

### Step 2: TAM を算出

```
対象ユーザー総数 = [セグメント人口]
TAM = 対象ユーザー総数 × ARPA

例:
  日本国内のフリーランスデザイナー: 約30万人
  ARPA: ¥12,000/年
  TAM = 300,000 × 12,000 = ¥3.6B（36億円）
```

### Step 3: SAM を算出

TAM からフィルタリング:

```
SAM フィルター:
  - 地理的制約: 日本語圏のみ → TAM の 100%
  - 技術リテラシー: ツールを日常的に使う層 → 70%
  - 課題認識: この問題を認識している層 → 50%

SAM = TAM × 0.70 × 0.50 = ¥1.26B（12.6億円）
```

### Step 4: SOM を算出

SAM からさらに現実的な獲得可能量を算出:

```
SOM の算出要素:
  - 初期チャネルリーチ: SAM の 5%
  - コンバージョン率: リーチした人の 3%
  - 1年目の想定顧客数: SAM × 0.05 × 0.03 = [人数]

SOM = 想定顧客数 × ARPA

例:
  リーチ: 12.6億 × 5% = 6,300万円分の市場にリーチ
  コンバージョン: 6,300万 × 3% = 189万円
  SOM = ¥1.89M（約190万円）
```

## 注意事項

- ボトムアップの数値は **参考推定値** であり、実際の市場環境により大きく変動する
- 重要な投資判断には一次情報（業界レポート・ユーザーインタビュー）を併用すること
- 成長率の前提は楽観的になりがちなので、保守的シナリオも作成すること
