# コンテンツ SEO 戦略テンプレート

## 概要

プロダクトの検索エンジン経由での発見性を高めるためのコンテンツ戦略。
MVP 段階ではミニマルに、成長期に拡充する。

## 1. キーワードリサーチ

### キーワード分類

| カテゴリ | 説明 | 例 |
|----------|------|-----|
| ブランドキーワード | プロダクト名 | "[プロダクト名]" |
| 課題キーワード | ユーザーの課題 | "[課題] 解決方法" |
| 機能キーワード | 主要機能 | "[機能名] ツール" |
| 比較キーワード | 競合比較 | "[プロダクト名] vs [競合名]" |
| ハウツーキーワード | 使い方 | "[カテゴリ] やり方" |

### キーワードマトリクス

| キーワード | 検索ボリューム | 難易度 | 意図 | 優先度 |
|-----------|-------------|--------|------|--------|
| | 高/中/低 | 高/中/低 | 情報/商用/トランザクション | A/B/C |
| | | | | |
| | | | | |

### 検索ボリューム調査方法

- Google キーワードプランナー（Google Ads アカウント必要）
- Ubersuggest（無料枠あり）
- Google Trends（トレンド比較）
- Google サジェスト（検索窓に入力して候補を確認）

## 3. 内部リンク構造

### ピラーページ戦略

```
ピラーページ（メインテーマ）
├── クラスターページ1（サブトピック）
├── クラスターページ2（サブトピック）
├── クラスターページ3（サブトピック）
└── クラスターページ4（サブトピック）
```

- ピラーページ: 包括的なガイド（2,000語以上）
- クラスターページ: 特定トピックの詳細（800-1,500語）
- 全クラスターページからピラーページへリンク

### サイト構造

```
/                    → メインLP
/features            → 機能紹介
/pricing             → 料金プラン
/blog/               → ブログ一覧
/blog/[slug]         → 個別記事
/docs/               → ドキュメント
/docs/getting-started → 入門ガイド
```

## 4. テクニカル SEO チェック

| 項目 | 確認方法 | 基準 |
|------|---------|------|
| タイトルタグ | 各ページに固有のタイトル | 30-60文字 |
| メタディスクリプション | 各ページに固有の説明 | 120-160文字 |
| H1 タグ | 各ページに1つのみ | ページ内容を反映 |
| 画像 alt テキスト | 全画像に設定 | 内容を説明する |
| URL 構造 | 短く意味のある URL | /blog/seo-tips |
| 正規 URL (canonical) | 重複ページに設定 | <link rel="canonical"> |
| モバイル対応 | レスポンシブデザイン | Google Mobile-Friendly Test |
| ページ速度 | Core Web Vitals | LCP < 2.5s |
