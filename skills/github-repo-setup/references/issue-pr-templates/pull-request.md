# Pull Request テンプレート

## ファイル配置先

`.github/PULL_REQUEST_TEMPLATE.md`

## テンプレート内容

```markdown
## 概要

<!-- この PR で何を変更したか、簡潔に説明してください -->

## 関連 Issue

<!-- closes #123 -->

## 変更種別

- [ ] バグ修正（既存機能の不具合修正）
- [ ] 新機能（後方互換性のある新機能追加）
- [ ] 破壊的変更（既存機能に影響する変更）
- [ ] ドキュメント更新
- [ ] リファクタリング（機能変更なし）
- [ ] テスト追加・修正
- [ ] CI/CD 変更

## 変更内容

<!-- 具体的な変更内容を箇条書きで記載してください -->

- 変更1
- 変更2
- 変更3

## テスト

<!-- どのようにテストしたかを記載してください -->

- [ ] ユニットテスト追加/更新
- [ ] E2E テスト追加/更新
- [ ] ローカルで動作確認済み

## スクリーンショット（UI変更の場合）

<!-- Before / After のスクリーンショットを添付してください -->

| Before | After |
|--------|-------|
| | |

## チェックリスト

- [ ] コードが ESLint/Prettier に準拠している
- [ ] TypeScript の型エラーがない
- [ ] テストが全て通過する
- [ ] ドキュメントを更新した（必要な場合）
- [ ] セキュリティへの影響を確認した

## レビューで特に見てほしい点

<!-- レビュアーに特に確認してほしい箇所があれば記載してください -->
```
